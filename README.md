# Blinkit Sales Analytics Project 

##  Project Overview
This project performs complete end-to-end sales analysis on Blinkit-style grocery sales data using Python, SQL, RFM segmentation, Machine Learning clustering, and Time-Series Forecasting.

## Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- RFM Customer Segmentation
- K-Means Clustering
- Customer Lifetime Value (CLV)
- Time-Series Sales Forecasting
- Business Insights & Visualization
